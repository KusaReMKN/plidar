# plidar

プログラムは xterm から実行される必要があります。

[KusaReMKN/jeedo]: https://github.com/KusaReMKN/jeedo
[test/lidar_test.c]: https://github.com/KusaReMKN/jeedo/blob/main/test/lidar_test.c

## sample.txt について

サンプルファイルが sample/sample.txt に含まれています。
サイズに注意してください。
